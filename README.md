# Hi there! I'm Roheemah 👋

## 🚀 About Me
Blockchain developer with a deep understanding of Ethereum, EVM architecture, and decentralized systems. Skilled in designing, writing, testing, and deploying secure smart contracts using Solidity and Rust, and integrating them with on-chain applications through Ethers.js, Wagmi, and Viem. Experienced in developing ERC standards (ERC20, ERC721), multi-sig wallets, auctions, staking, and NFT marketplaces. Passionate about security, zero-knowledge, and protocol-level innovation, with a proven record of writing clean, auditable code and contributing to open-source projects.

My current focus is on **Ethereum core contributions** — I work across the consensus layer (ethereum/consensus-specs, Lighthouse) and execution layer (Nethermind), with a strong interest in protocol security, sync mechanisms, and client implementation.
 
I'm also a trainer and mentor: I've served as a training officer at **Web3bridge**, consulted on the protocol math curriculum there, and mentor junior developers at **Dev3pack**.
